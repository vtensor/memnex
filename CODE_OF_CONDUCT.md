# Code of Conduct

The Memnex project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1.

In short: be respectful, be constructive, assume good intent. We expect contributors to welcome newcomers, accept feedback graciously, and focus criticism on ideas rather than people. Behaviour that makes the project unwelcoming for others is not tolerated.

## Reporting

If you experience or witness behaviour that violates this code, please report it privately to **vikramdev.iitd@gmail.com**. Reports are handled confidentially. The maintainers will respond within 7 days and take whatever action they judge appropriate, up to and including removing contributors from the project.

## Scope

This applies to all project spaces — GitHub issues and PRs, discussion threads, any official chat or call, and any public space where someone is representing the project.

For the full text of the standards, scope, enforcement, and attribution, see the official Contributor Covenant 2.1 at the link above.
